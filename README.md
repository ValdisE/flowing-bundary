# flowing-bundary

## Install

```bash
npm install

# or

yarn install
```

## Serve

```bash
npm run dev

# or

yarn dev
```

Then Open In Browser

```bash
   ┌─────────────────────────────────────────────────┐
   │                                                 │
   │   Serving!                                      │
   │                                                 │
   │   - Local:            http://localhost:3000     │
   │                                                 │
   └─────────────────────────────────────────────────┘
```
